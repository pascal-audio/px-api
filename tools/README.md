# PX API Tools

## Quick Install

Run the install script for your platform:

**Linux/macOS:**
```bash
./install.sh
```

**Windows:**
```cmd
install.bat
```

## What's Included

- **japi_cli/** - Python command-line interface with full API support
  - See `japi_cli/README.md` for complete documentation

## Manual Setup

If you prefer not to use the install scripts, see `japi_cli/README.md` for manual installation instructions using `uv` or `pip`.
